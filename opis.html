Opis ideje, ki jo aplikacija podpira

Aplikacija Asteoridi, je namenjena pregledu predmetov, ki so v zemljini orbiti, na določeni dan. Ko uporabnik izbere željeni datum, se mu bo prikazala tabela s naslednjimi podatki: ime asteroida, ali je lahko nevaren, kakšna je njegova velikost, za koliko metrov je asteroid zgrešil Zemljo in načrtovan dan, ko bo najbližje Zemlji. Aplikacija tudi vsebuje segment z imenom Dnevne vesoljske novice, kjer uporabnik lahko spremlja dogajanje iz vesolja. Dnevne vesoljske novice se dnevno posodabljajo, s pomočjo NASA API. Aplikacija ima za ozadje premikajoče zvezde, katerim v spodnjem levem kotu lahko spemenimo hitrost. Aplikacija je predvsem namenjena radovednežem, ki se zanimajo za podatke asteroidov.

Uporabljeni zunanji viri podatkov

Za svoj zunanji API sem z metodo GET pridobila podatke iz NASA API-ja (URL: https://api.nasa.gov/). Uporabljen je API z imenom Asteroids-NeoWs. API sem uporabila v datoteki koda.js z neposrednim fetch klicem API . NASA API nam omogoča dostop po podatkov, ki so navedeni v tabeli in Dnevnih vesoljskih novicah. Namen API-ja je, da ima aplikacija dostop do podatkov, ki so dnevno posodobljeni.

Master/detail načrtovalski vzorec

Ob izbiri datuma se pokaže tabela, ki prikazuje podatke asteroidov na izbrani dan. Če spremenite datum se bo tabela posodobila.

(PS. Za prikaz tabele je potrebno počakati nekaj trenutkov, odvisno od zmogljivosti vaše opreme.)

Uporabljena tehnika za vizualizacijo podatkov

Za vizualizacijo podatkov sem uporabila Google Charts Razlog za uporabo Google Charts je enostavna in poznana uporaba. Za ozadje pa sem oporabila, zvezde katerim se lahko spreminja hitrost. Uporabila sem jih, saj se ujemajo z tematiko aplikacije.
